n8n-oauth

Última actualización: 10 de enero de 2026

Propósito: Esta página describe la aplicación n8n-oauth, cuyo objetivo es que usuarios autorizados puedan conectarse a nuestro servidor n8n de forma rápida, segura y didáctica.

¿Qué hace esta aplicación?

n8n-oauth facilita el proceso de inicio de sesión y vinculación de cuenta para acceder a un servidor n8n gestionado por Secuoya. Está pensada para onboarding de nuevos usuarios cuando se les concede acceso al entorno (por ejemplo, colaboradores o clientes autorizados).

¿Quién puede usarla?

  • Usuarios que hayan sido invitados o autorizados por el administrador del servidor n8n.
  • Personas que necesiten un proceso guiado para completar el acceso.

¿Qué datos se solicitan y por qué?

Dependiendo del proveedor de inicio de sesión, la app puede solicitar datos como nombre y correo para identificar al usuario y habilitar el acceso correcto. El objetivo es solicitar solo lo necesario para autenticar al usuario.

Para detalles completos, revisa: Políticas de privacidad.

Pasos para conectarte (guía rápida)

  1. Recibe una invitación o confirmación de acceso al servidor n8n.
  2. Entra a esta página y selecciona “Iniciar sesión / Conectar”.
  3. Completa el inicio de sesión con tu proveedor (OAuth).
  4. Revisa los permisos solicitados y confirma solo si estás de acuerdo.
  5. Serás redirigido al servidor n8n con tu acceso habilitado.

Importante: Si no tienes acceso, o tu cuenta no está autorizada, el sistema puede bloquear el ingreso. En ese caso, contacta al administrador o a soporte.
