Condiciones del servicio

Aplicación: Secuoya n8n-oauth  ·  Última actualización: 10 de enero de 2026

Estas Condiciones regulan el uso de Secuoya n8n-oauth (la “Aplicación”), una interfaz diseñada para que usuarios autorizados puedan conectarse a un servidor n8n de forma rápida, didáctica y guiada.

1. Definiciones

  • “Aplicación”: Secuoya n8n-oauth.
  • “Servicio”: Acceso guiado y autenticación para ingresar a un servidor n8n autorizado.
  • “Usuario”: Persona invitada o autorizada para usar el Servicio.
  • “Servidor n8n”: Instancia de n8n gestionada por Secuoya Group o por un cliente autorizado.

2. Propósito y funcionamiento

La Aplicación existe para facilitar el inicio de sesión y la activación del acceso a n8n cuando un Usuario recibe permiso para usar el Servidor n8n. La Aplicación busca que el Usuario entienda qué está autorizando y para qué se usará (transparencia).

La Aplicación no se ofrece como un producto masivo; se orienta a usuarios previamente autorizados por Secuoya o por el administrador del Servidor n8n.

3. Elegibilidad y acceso

  • Debes tener autorización expresa para acceder al Servidor n8n.
  • Eres responsable de mantener la confidencialidad de tu cuenta y/o credenciales.
  • Si representas a una organización, declaras que tienes autorización para usar el Servicio en su nombre.

4. Uso permitido y restricciones

Uso permitido:

  • Autenticarse para ingresar al Servidor n8n al que fuiste invitado.
  • Consultar información del estado de conexión y guías de uso.

Restricciones:

  • No usar el Servicio para acceso no autorizado, suplantación o eludir controles de seguridad.
  • No intentar extraer secretos, tokens o credenciales de otros usuarios.
  • No realizar pruebas de carga/ataques (DDoS), escaneo o actividades maliciosas.
  • No usar el Servicio para fines ilegales o que vulneren derechos de terceros.

5. Autenticación (OAuth) y permisos

La Aplicación puede utilizar OAuth con proveedores de identidad para autenticarte. Debes revisar la pantalla de consentimiento y aceptar únicamente si estás de acuerdo.

El Servicio procurará solicitar solo los permisos mínimos necesarios para habilitar el acceso al Servidor n8n (por ejemplo, identidad básica).

6. Tratamiento de datos y privacidad

El tratamiento de información personal se rige por nuestra Política de Privacidad. La Política explica qué datos se recopilan, para qué se usan, cómo se almacenan, por cuánto tiempo y cómo solicitar eliminación. 7. Seguridad

Implementamos medidas razonables para proteger el Servicio. Aun así, ningún sistema es 100% infalible. Debes notificar inmediatamente cualquier uso no autorizado o incidente.

8. Disponibilidad y soporte

El Servicio se ofrece “tal cual” y puede tener ventanas de mantenimiento. Cuando sea posible, informaremos con antelación de cambios relevantes.

9. Suspensión o terminación

Podemos suspender o revocar el acceso si detectamos uso indebido, riesgo de seguridad o incumplimiento de estas Condiciones, o si el administrador del Servidor n8n retira tu autorización.

10. Cambios a estas condiciones

Podemos actualizar estas Condiciones. Si el cambio afecta materialmente el uso del Servicio, lo indicaremos en esta página y, cuando corresponda, por un canal de contacto disponible.
